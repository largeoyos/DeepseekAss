# 使用说明

## 一、三种模式

| 模式 | 说明 |
|------|------|
| 🎭 角色扮演 | 让 AI 扮演某个角色和你对话（比如扮演一个骑士、一只猫、一个名人） |
| 📚 小说写作 | 让 AI 帮你写小说，支持书架管理、章节续写、自动记剧情 |
| 📄 续写小说 | 给你已有的文章（.txt 或 .md 文件），AI 接着往下写 |

## 二、界面布局

窗口分左右两栏：

- **⬅️ 左边**是控制区：选模式、选模型、调参数、填写角色/小说设定
- **➡️ 右边**是聊天区：上面显示对话内容，下面输入消息

## 三、每个按钮是干什么的

### 3.1 全局通用

**📌 聊天模式（下拉框）**
切换三种模式，切换后左边面板会显示对应模式的设定项。

**🧠 模型选择（下拉框）**
选 `deepseek-v4-flash`（快）或 `deepseek-v4-pro`（强）。

| 参数 | 作用 | 范围 |
|------|------|------|
| 🌡️ 温度（滑块） | 控制 AI 创造力，越高越天马行空 | 0~2 |
| 🎯 top_p（滑块） | 也是控制创造力的，一般保持默认 | 0~1 |
| 🔄 freq_p（滑块） | 频率惩罚，越高越不重复用词 | -2~2 |
| 📏 max_tk（数字框） | AI 一次最多输出的 token 数，写长文时调大 | 1~300000 |

**🗑️ 清除对话**
清空聊天记录回到初始页面。只有角色扮演模式有这个按钮。

### 3.2 聊天区

**✏️ 输入框**
打字的地方。按 Ctrl+Enter 发送。

**🚀 发送按钮**
发消息。小说模式下如果勾了"章节续写模式"，点发送会直接生成下一章而不是聊天。

### 3.3 对话历史（仅角色扮演模式）

| 按钮/控件 | 作用 |
|-----------|------|
| 💾 保存当前对话 | 把现在的对话存起来，以后可以再打开 |
| 📋 对话历史列表（下拉框） | 选一个以前存过的对话 |
| 📂 加载 | 打开选中的对话，自动切到当时的模式和模型 |
| 🗑️ 删除 | 删掉选中的对话，不能恢复 |
| 📤 导出对话 | 导出成 TXT/Markdown/HTML/Word 文件 |

### 3.4 角色扮演设定

| 控件 | 说明 |
|------|------|
| 👤 角色描述 | 要 AI 扮演谁，比如"一位傲娇的中世纪骑士" |
| 🌍 故事背景 | 故事发生在什么世界，比如"中世纪欧洲，有魔法" |
| 💬 回复方式（单选） | 角色回答（第一人称"我"）或 旁白描述（第三人称"他/她"） |
| ✅ 应用设定 | 把上面填的内容写入 AI 指令，保留聊天记录不清除 |

### 3.5 小说写作面板

| 按钮/控件 | 作用 |
|-----------|------|
| 📖 书架（下拉框） | 选一本已创建的小说 |
| ➕ 新建 | 创建新小说，输入标题即可 |
| 🗑️ 删除 | 删除选中的小说（包括所有章节，不可恢复） |
| ✏️ 标题 | 当前小说名字 |
| 📝 章节 | 下一章的标题 |
| ℹ️ 章节信息（只读） | 显示当前有多少章 |
| ✅ 章节续写模式（复选框） | 勾上后发送消息 = 生成下一章；不勾 = 正常聊天 |
| 👤 主角设定 | 写主角背景/性格/外貌，改了自动保存 |
| 🌍 世界观/背景故事 | 写时代/世界设定，改了自动保存 |
| ✍️ 写作要求 | 本章特别要求，改了自动保存 |
| 📝 本章情节输入（可选） | 你想在这一章发生的具体情节，不填 AI 自己编 |
| 🔢 字数 | 设定目标字数，默认 2000 |

**🚀 生成下一章**
核心按钮，点一下 AI 自动做以下事情：
1. 📦 收集你填的设定
2. 📖 读取前面章节的剧情摘要（短篇全读，长篇压缩早期章节）
3. 📕 读世界书（已提取的角色、地点、规则）
4. 🤖 AI 写一章
5. 💾 自动保存文件
6. 📋 提炼这章摘要记下来
7. 📏 字数不够 80% 自动补充
8. 🔄 自动更新世界书（提取新人物、地点）

| 按钮 | 作用 |
|------|------|
| 💾 保存小说设定 | 手动保存一次主角/世界观/写作要求 |
| 📂 加载小说设定 | 从文件加载当前书的设定到编辑框 |

**⚙️ 章节管理**
打开版本管理窗口，里面的按钮：

| 按钮 | 作用 |
|------|------|
| 👁️ 预览 | 看某一章的完整内容 |
| ⭐ 设为活跃（计入剧情） | 把某版本设为正式版（只有活跃版参与剧情摘要），可选是否重建摘要 |
| 🔁 重新生成 | 用当前设定重写某章，存为新版本（v+1），不覆盖旧的 |
| 🗑️ 删除此版本 | 删掉某版本，不能删活跃版本 |

| 按钮 | 作用 |
|------|------|
| 📖 世界书 | 查看/编辑 AI 自动提取的角色、地点、设定规则 |
| 📄 分段摘要 | 选一个文件，AI 自动分段并为每段写摘要 |
| 📋 导出格式（下拉框） | 选 TXT/Markdown/HTML/Word |
| 📤 导出当前章节 | 把最新一章导出来 |
| 📚 导出全书 | 把所有章节一起导出来 |

### 3.6 续写小说面板

| 按钮/控件 | 作用 |
|-----------|------|
| 📂 源文档 → 浏览 | 选一个 .txt/.md 文件作为续写来源 |
| 📁 或指定文件夹 → 浏览 | 选一个文件夹，程序把所有文本文件拼起来 |
| ✍️ 续写要求 | 对续写提风格/视角/节奏等要求 |
| 🔢 字数 | 续写目标字数 |
| 📝 续写剧情（可选） | 续写内容走向，不填 AI 自己决定 |

**🔍 分析源文档**
先分析原文提取设定和剧情，弹对话框给你看。可以选：
- 🎲 让 AI 建议几个发展方向，选一个续写
- ✏️ 自己写剧情走向续写

| 按钮 | 作用 |
|------|------|
| ▶️ 开始续写（跳过分析） | 不分析直接写，省一步 |
| 📤 导出格式 + 导出章节/全书 | 跟小说面板一样 |

## 四、自动帮你做的事

1. 💾 修改主角/世界观/写作要求时，自动保存到文件
2. 📏 章节字数不够目标 80% 时，自动补充内容
3. 📖 每生成一章，自动提取新角色新地点，更新世界书
4. 📋 同一章可以保留多个版本，切换到哪个版本就算哪个
